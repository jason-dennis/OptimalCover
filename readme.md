# OptimalCover 🛜

> Smart City WiFi infrastructure optimizer — *in progress*

Determines optimal placement of public WiFi access points across urban zones,
formulated as a capacitated Facility Location Problem (FLP).

## Current Progress
- [x] Overpass API integration (real urban data)
- [x] JSON infrastructure parser (C++)
- [ ] FLP solver
- [ ] Visualization

## Tech Stack
- C++
- Overpass API / OpenStreetMap
- Mathematical optimization (FLP)
